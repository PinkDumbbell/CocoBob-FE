# Petalog 펫탈로그
### 반려동물을 위한 사료 추천 및 건강 관리 애플리케이션

### 펫탈로그란?

 각자 개성이 넘치는 사랑스러운 **반려동물에게 맞춤 정보**를 제공해드립니다!  
 현재 건강상태, 생애 주기, **AAFCO**(미국 사료 협회) 영양 기준을 만족하는 사료를 추천해 드립니다  
 하루의 생활을 기록해보세요. 산책, 몸무게, 급여, 일기, 특이사항을 작성하고 더욱더 정확한 제품을 추천받아보세요.
